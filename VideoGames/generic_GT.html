Texture Sheets

A furnished textured environment set for video games

By digitally painting 3D models, an environment set and several repeatable textures were created for use in video games instead of lms or animation.

*The 3D model in this environment set was provided from my insttructor in the texture class in Game Design in Academy of Art University.

Tileable textures

It is considered quite challenging to manually paint a texture and achieve a realistic style. Usually, artists combine photos with hand-painted brush strokes to create realistic textures; however, the task was to create tileable textures for video games using only hand painting. I am pretty satised with the result and hope that in the future my texture drawing skills can be improved further.
